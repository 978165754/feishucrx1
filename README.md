## 发布
请先npm run build，连同dist目录一起提交，然后再填写表单：
[共享表单](https://feishu.feishu.cn/share/base/form/shrcnGFgOOsFGew3SDZHPhzkM0e)


可以选择多为表格某个数据表内的指定的多个列（字段），把他们都合并成1个文本内容，然后插入指定的某个数据表的指定某一列。

1.可以指定来源表和目标表格
2.可以指定选择多个列，按选则的顺序获取文本内容并拼接
3.可以指定拼接的符号，例如逗号 冒号等
4.可以指定插入到某个表的某个字段内
